# Portafolio Profesional - Diego Olivera 🚀

Portafolio web personal estático diseñado para mostrar proyectos, habilidades técnicas y experiencia en el desarrollo de software. Construido bajo los estándares de accesibilidad (a11y) y metodologías modernas de desarrollo frontend.

🔗 **[Visitar el sitio en vivo](https://ryzenz10.github.io)** ## 🛠️ Pila Tecnológica (Tech Stack)

Este proyecto fue desarrollado utilizando una arquitectura nativa para garantizar tiempos de carga mínimos y control total sobre el DOM:

* **HTML5 Semántico:** Estructura optimizada para motores de búsqueda (SEO) y lectores de pantalla.
* **CSS3 (Variables, Flexbox & Grid):** Diseño 100% responsivo bajo el enfoque *Mobile First*. Arquitectura de estilos basada en la metodología **BEM** (Block, Element, Modifier).
* **JavaScript (Vanilla ES6+):** Interactividad del menú de navegación optimizada sin dependencias externas.
* **Formspree:** Integración BaaS (Backend as a Service) para la gestión asíncrona del formulario de contacto.

## ⚙️ Características Destacadas

* **Rendimiento Optimizado:** Estructura de red ligera auditada con Google Lighthouse.
* **UI/UX:** Paleta de colores de alto contraste, tipografía fluida y microinteracciones en componentes críticos.
* **Despliegue Continuo (CI/CD):** Alojado en GitHub Pages con actualizaciones automatizadas a través de control de versiones.

## 📂 Estructura del Proyecto

```text
├── assets/                  # Imágenes, íconos y documento CV (PDF)
├── index.html               # Estructura semántica base
├── style.css                # Estilos globales y maquetación de componentes
├── script.js                # Lógica de interactividad UI
└── README.md                # Documentación del proyecto
```
